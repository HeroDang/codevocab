# CodeVocab
Android app (Kotlin, XML) - học từ vựng IT

## Quick start
1. Clone repo
2. Add local.properties (sdk path)
3. Open in Android Studio -> Sync Gradle -> Run

## Branching
- main: stable
- dev: integration
- feature/*: task-specific

## Hoạt động nhóm
- Tạo PR từ feature -> dev
- 1 reviewer trước khi merge
